# 如何看待国产大模型与海外大模型在 agent 能力上的差距？

过去半年明显感觉 GPT-4 / Claude 在工具调用、长任务稳定性上拉开差距。
国产主要卡在两个点：

1. 高质量工具使用训练数据不足——开源 dataset 里真实 agent 轨迹很少
2. RL 后训练阶段对 tool-use 的 reward shaping 不够精细

但短板会追回来。阿里、百度、月之暗面都在各自补。估计 6-12 个月缩到可接受范围。
