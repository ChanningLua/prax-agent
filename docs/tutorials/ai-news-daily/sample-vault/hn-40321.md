# The pitfalls of async agent loops without observability

Ran a 30-minute agent task last night that silently burned $14 because the inner loop re-called a broken tool 200 times. No metrics dashboard, no alerts.

Lesson: even if your agent framework has "self-correction," add per-tool call count + cost tracking. The failure mode without telemetry is not "AI does bad thing" but "AI does pointless thing forever until budget dies."
